# express-boilerplate

What you need to run this code :

1. Node
2. NPM 
3. MongoDB, Mongoose

How to run this code :
1. Make sure MongoDB ATLAS is running
2. Clone this repository
3. Update config/default.js with your test values for mongoDB
4. Open command line in the cloned folder,
To install dependencies, run npm install
5. To run the application for development, run npm start
6. Open localhost:5000 in the browser
